# Lista-De-Materias-Re-Redes-Gpon
Rastreador de Projetos GPON/FTTH feito em Python para automação na engenharia da Gigalink. O sistema lê arquivos KMZ/KML do Google Earth e gera o quantitativo e o orçamento de materiais instantaneamente.  Identifica bitolas de cabos por cor.Destrincha caixas HUB/CEO (Kits, Splitters e Bandejas).  Exportação em CSV.
